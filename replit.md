# Türk Ligi Discord Maç Botu

## Overview

Discord bot for simulating Turkish football league matches. Teams have base ratings, AI analyzes uploaded tactic .txt files (0–6 boost), and matches are simulated using a GPR (Gerçek Performans Reytingi) formula plus randomness. Match narratives are generated by Gemini AI.

## Stack

- **Bot**: discord.js v14, slash commands (Turkish)
- **AI**: Gemini 2.5 Flash via Replit AI Integrations (no own API key)
- **Database**: PostgreSQL + Drizzle ORM
- **Runtime**: Node.js 24, esbuild bundle
- **HTTP**: Express 5 (just a /api healthcheck endpoint)

## GPR Formula

`GPR = Base Rating + Home Bonus (+3 / 0) + AI Tactic Boost (0..6)`

Match outcome uses a Poisson distribution seeded with expected goals derived from GPR difference. Random events (cards, scorers, assisters) are weighted by player ratings and positions.

## Schema (lib/db/src/schema)

- `teams` — name, shortName, baseRating, color, season stats (W/D/L/GF/GA/Pts)
- `players` — teamId, name, position (GK/DEF/MID/FWD), rating, goals, assists, cards, appearances
- `tactics` — one per team: fileName, content, AI tacticScore, formation, analysis
- `matches` — full simulation history with events JSON
